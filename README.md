Job Portal Registration Website

Project Overview

The Job Portal Registration Website is a **static web application** developed using HTML, CSS, and JavaScript. It provides a modern and responsive interface for user registration and login. This project focuses on frontend development, user interface design, and client-side form validation without using a database or backend technology.

Technologies Used

* HTML5
* CSS3
* JavaScript

IDE Used

* Visual Studio Code
* Live Server

Features

* User Registration Form
* User Login Page
* Responsive Design
* Modern Dark UI
* Client-side Form Validation
* Password Confirmation Validation
* Resume Upload Interface
* Interactive Buttons and Hover Effects

Project Limitations

* This is a **static website**.
* No database is connected.
* User information is **not stored permanently**.
* Login is for demonstration purposes only and does not authenticate users through a server or database.

Software Requirements

* Windows 10/11
* Visual Studio Code
* Google Chrome
* Live Server Extension

Hardware Requirements

* Intel Core i3 or above
* 4 GB RAM or more
* 20 GB Free Storage

How to Run the Project

1. Open the project folder in Visual Studio Code.
2. Ensure all project files are in the same folder.
3. Open `registration.html` or `login.html`.
4. Click **Go Live** using the Live Server extension or open the HTML files in a web browser.
